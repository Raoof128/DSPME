# Code of Conduct

We follow the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In brief:
- Be respectful and inclusive.
- Provide constructive feedback.
- Report unacceptable behavior to the maintainers via security contact in `SECURITY.md`.

Maintainers will enforce this code and may remove contributions that violate these standards.
